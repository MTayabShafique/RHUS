# RHUS
RFID â€“ based Utility System for Hotels is a system to monitor electricity consumption of the hotel and provide access for customized services to the customers. There are two types of users for this system, Manage and Receptionist. Manager will be able to monitor electricity of the rooms of hotel as well as of the whole building. Receptionist will be able to add customers to the hotel and provide them access to customized services in the room with the help of RFID cards.
